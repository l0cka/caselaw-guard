# Security Policy

## Reporting a Vulnerability

Please report suspected security vulnerabilities through GitHub Security Advisories for this repository:

https://github.com/l0cka/caselaw-guard/security/advisories/new

Avoid opening public issues for suspected vulnerabilities. Include the affected version or commit, reproduction steps, impact, and any suggested remediation.

## Supported Versions

This project is currently pre-1.0. Security fixes are made on the `main` branch and included in the next tagged release.
